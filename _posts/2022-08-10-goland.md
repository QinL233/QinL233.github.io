# 一、安装Goland环境

## 创建项目
```
go env

go env -w GO111MODULE=on

go env -w GOPROXY=https://goproxy.cn,direct

go mod init [项目名称]
```

# 二、Goland开发工具设置

## GOROOT设置
1、使用go env 查看GOROOT位置

2、编辑C:\Program Files\Go\src\runtime\internal\sys\zversion.go文件末行增加
```
const TheVersion = `go1.17.5`
```
