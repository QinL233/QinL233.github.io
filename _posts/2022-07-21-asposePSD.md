# 二、修改jar

反编译工具：JbyteModify

class操作组件：javassist
https://www.jianshu.com/p/7bed2ad09230


```
ClassPool.getDefault().insertClassPath("xxx.jar");
CtClass c = ClassPool.getDefault().getCtClass("xxx.class");
CtMethod[] as = c.getDeclaredMethods("method");
as[2].insertAfter("{return 0;}");
c.writeFile("D:\\");

```
